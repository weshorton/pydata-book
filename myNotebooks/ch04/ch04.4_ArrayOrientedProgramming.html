Expressing Conditional Logic as Array Operations

Consider the expression x if condition else y

The function numpy.where is a vectorized version of this expression.

Consider the following arrays:

In[]:
xarr = np.array([1.1, 1.2, 1.3, 1.4, 1.5])
yarr = np.array([2.1, 2.2, 2.3, 2.4, 2.5])
cond = np.array([True, False, True, True, False])
print(xarr)
print(yarr)
print(cond)

[1.1 1.2 1.3 1.4 1.5]
[2.1 2.2 2.3 2.4 2.5]
[ True False  True  True False]

And imagine we want to grab from xarr at all the indices that cond == True and grab from yarr otherwise.

We can do this with the following list comprehension:

In[]:
result = [(x if c else y) for x, y, c in zip(xarr, yarr, cond)]
result
Out[]:
[1.1, 2.2, 1.3, 1.4, 2.5]

The above is slow and also only works for 1-d arrays. where will just match them all up and evaluate:

In[]:
result2 = np.where(cond, xarr, yarr)
result2
Out[]:
array([1.1, 2.2, 1.3, 1.4, 2.5])

So this is a vectorized ifelse? You don't have to provide arrays to the second and third arguments.

Below, we make an array, and then we replace values based on a condition (in this case positive values -> 2 and negative values -> -2)

all is similar, but checks that all values are True.

Sorting

sort method will sort arrays in place (just like lists)

If you have a multi-d array, provide the axis argument to sort each 1-d array independently.

Using the function np.sort instead of the method returns a copy of the original array.

Unique and Other Set Logic

Numpy has some set operations that act on 1-d arrays. They're generally faster than the base python alternatives and return arrays instead of lists.

unique and in1d are shown below, here is a table with more:

In[]:
# Array of names
names = np.array(["Will", "Bob", "Joe", "Bob", "Will", "Joe", "Joe"])
names
Out[]:
array(['Will', 'Bob', 'Joe', 'Bob', 'Will', 'Joe', 'Joe'], dtype='<U4')
In[]:
# Use unique method to retun an array
np.unique(names)
Out[]:
array(['Bob', 'Joe', 'Will'], dtype='<U4')
In[]:
# Default python is a list
set(names)
Out[]:
['Bob', 'Joe', 'Will']
In[]:
# Array of integers
ints = np.array([3, 3, 3, 2, 2, 1, 1, 4, 4])
np.unique(ints)
Out[]:
array([1, 2, 3, 4])
In[]:
# Array of values
values = np.array([6, 0, 0, 3, 2, 5, 6])
values
Out[]:
array([6, 0, 0, 3, 2, 5, 6])
In[]:
# Check if each element in values is in the array [2, 3, 6]
np.in1d(values, [2, 3, 6])
